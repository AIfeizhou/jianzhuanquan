## 🚀 部署信息

### 系统要求
- **操作系统**: Windows 10/11, macOS, Linux
- **Node.js**: >= 18.0.0
- **Python**: >= 3.7
- **内存**: >= 4GB RAM
- **存储**: >= 2GB 可用空间

### 服务端口
- **后端API**: 3000
- **前端应用**: 3001
- **上传目录**: ./uploads
- **临时文件**: ./temp

### 环境配置
- 配置文件: `config.env`
- AI服务: 豆包AI API
- 代理支持: Rubiz代理服务
- 安全设置: Helmet + CORS
